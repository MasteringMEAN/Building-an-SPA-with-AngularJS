Building Single Page Web Apps with AngularJS

  • Section 1: Introduction to AngularJS

    In this section, we're going to introduce the basic concepts that you'll need to grasp in order to start developing with AngularJS. We'll talk about its advantages and applications and overview its architecture while reviewing the Model–view-controller pattern. Moreover, we will discuss what MVC really means from the perspective of a frontend framework.

    • The Course Introduction and Objectives
    • Prerequisites
    • Understanding the Basics of AngularJS and the MVC Architecture

    Browse Videos

  • Section 2: Your First AngularJS App

    In this section, we're going to dive head first into AngularJS development. We'll build a very basic application (such as, a to-do list), including how to use a boilerplate to kick off your app and review the basic mechanics of the framework in the process such as the use of directives, scopes, modules, and much more.

    • Angular-seed
    • Building a “To-do App” - Part 1
    • Building a “To-do App” - Part 2
    • Building a “To-do App” - Part 3
    • Building a “To-do App” - Part 4

    Browse Videos

  • Section 3: Building a Complex Single Page Application

    In this section, we're going to build our second application. This time, we're going for something way more complex (such as a movie database application), which will give us the chance to introduce many other concepts of the framework, such as routing, filters, services, factories, and partial views.

    • Building a Complex Single Page Application - Part 1
    • Building a Complex Single Page Application - Part 2
    • Building a Complex Single Page Application - Part 3
    • Building a Complex Single Page Application - Part 4
    • Building a Complex Single Page Application - Part 5
    • Building a Complex Single Page Application - Part 6

    Browse Videos

  • Section 4: Directives and Filters

    In this section, we're going to take an in-depth look at directives and filters. We'll review both the basic and advanced directives available in the framework, and then we're going to learn how to write our own custom directives. Then, we will do the same for filters.

    • AngularJS Directives
    • Writing Custom Directives
    • AngularJS Filters

    Browse Videos

  • Section 5: Services and Factories

    In this section, we're going to take an in-depth look at the services and factories modules, which are fundamental to understand the integration of AngularJS with server-side technologies. We're going to find out more details on how to use asynchronous data requests through the $http and $resource services.

    • Writing Custom Services
    • AngularJS Services

    Browse Videos

  • Section 6: Testing in AngularJS

    In this last section, we're going to review the core concepts of how to perform proper test-driven development using AngularJS. We're going to write and run Karma unit tests with Jasmine and review the process of creating end-to-end tests with Protractor (using our movie database app as the groundwork).

    • Automated Testing in AngularJS
    • Unit Testing with Karma
    • E2E Testing with Protractor
